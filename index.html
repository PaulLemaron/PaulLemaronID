Project 2 · Tenwek Hospital

Hospital Laundry Training Program

15 modules · Practical labs · Video scripts · 1,500+ staff
🧺

Project Overview

This comprehensive training program was designed to equip every laundry staff member at Tenwek Hospital with the knowledge and skills to ensure patient safety through proper linen management. The program covers 15 modules spanning the entire laundry workflow: from receiving and segregation, through barrier washer operation, main washing, drying, finishing, dispatch, and quality management.

Project 4 · Instructional Design Concept

ADDIE Model Demonstration

Applying the ADDIE framework to instructional design
🧠

Project Overview

This project demonstrates my comprehensive understanding and application of the ADDIE (Analysis, Design, Development, Implementation, Evaluation) instructional design model. The demonstration shows how each phase of the model informs and shapes the creation of effective learning experiences, using a real-world case study.

ADDIE Instructional Design Case Study Needs Analysis Evaluation
1

Analysis

Needs assessment · Learner analysis · Context analysis

2

Design

Learning objectives · Storyboarding · Assessment design

3

Development

Content creation · Multimedia · Prototyping

4

Implementation

LMS deployment · Training · Rollout support

5

Evaluation

Formative & summative · Feedback · Iteration

📌 Key Takeaway: The ADDIE model provides a systematic, iterative approach to instructional design that ensures learning solutions are effective, efficient, and aligned with learner needs. This demonstration shows how each phase builds on the previous to create a comprehensive learning experience.
🔍 The Application: The ADDIE process was applied to the The Hospital Laundry Training project:
  • Analysis: Identified training gap for 1,500+ staff; needs analysis with hospital management
  • Design: 15 modules mapped to workflow; storyboarding; visual mockups
  • Development: Interactive HTML/CSS/JS content; Canva graphics; video scripts
  • Implementation: Canvas LMS deployment; staff onboarding support
  • Evaluation: Knowledge checks; practical labs; feedback mechanism

Project 5 · Master's Thesis · American University of Beirut

Data Science Bootcamp: Bridging the Skills Gap in Kenya

Project-Based Learning · Masterstudy LMS · Google Colab · GitHub
📊

Project Overview

This project is my Master's thesis at the American University of Beirut (AUB), addressing Kenya's persistent data science skills gap. The project involved designing, developing, and implementing a project-based online data science bootcamp to bridge the disconnect between theoretical university curricula and the practical competencies demanded by industry.

"Kenya's rapidly expanding digital economy is hampered by a persistent skills gap, particularly in data science, limiting innovation in critical sectors like agriculture and healthcare." — Thesis Abstract
